# Filetransfer Web applicatie

Wij maken een website waarmee je veilig zip bestanden kunt uploaden en downloaden.

## Omschrijving:
Voor het versturen van een bestand is de website gemaakt. Je kunt het uploaden en vervolgens met een link en code doorsturen. Met verschillende methodes wordt dit beveiligd. Het is vooral bedoeld om zip bestanden te delen. Je start op de homepage, en kunt kiezen tussen upload en download. Bij upload kun je een .zip bestand selecteren. Dit genereert een code en een link. Bij het downloaden kun je de link gebruiken of naar de website gaan. Daar kun je de code invullen om het bestand op te halen. Vervolgens kun je het downloaden en op je eigen systeem inzien.
