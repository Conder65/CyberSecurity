# Filetransfer Web applicatie

Wij maken een website waarmee je veilig bestanden kunt uploaden en downloaden.

## Omschrijving:
Voor het versturen van een bestand is de website gemaakt. Je kunt het uploaden en vervolgens met een link en code doorsturen. Met verschillende methodes wordt dit beveiligd. Het is vooral bedoeld om bestanden te delen. Je start op de homepage, en kunt kiezen tussen upload en download. Bij upload kun je een bestand selecteren. Dit genereert een code en een link. Bij het downloaden kun je de link gebruiken of naar de website gaan. Daar kun je de code invullen om het bestand op te halen. Vervolgens kun je het downloaden en op je eigen systeem inzien.

### Dag 3 updates:
Het inloggen is afgemaakt. Je kunt in de velden invullen wat nodig is, en in de database wordt een gehashed stukje weergegeven bij de wachtwoord. De controllers checken of er is ingelogd. Daarmee wordt voorkomen dat iedereen iets in de database kan laden. Ook zie je in eerste instantie de inlog en registratie, dus je zou niet meteen bestanden kunnen uploaden. Bij het inloggen zelf zijn er een aantal checks, om zeker te zijn dat alles juist wordt ingevuld. De front-end laat ook zien dat je een code nodig hebt bij download. Hoe dit goed geprogrammeerd wordt is voor een brainstorm sessie op een later moment. Dit willen wij op een goede manier afhandelen, zodat niet alles op de database willekeurig gedownload kan worden.


### Dag 4 updates 
Vandaag hebben wij een aantal afrondingstaken verdeeld. De front-end werd bijgewerkt en functioneel afgemaakt. Een logging systeem werd bedacht en aan de taken toegevoegd. Out of scope categorie werd toegevoegd voor alles dat fijn zou zijn, maar niet meer als verplichte toevoeging gedaan. Een logging systeem is toegevoegd voor wie, wanneer welk bestand up- of download.
